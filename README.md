# Ruby. Notas para Programadores 
## _Um compendio de dicas sobre como as coisas funcionam_

### Sobre
Isto é  uma tradução em forma de repositório do livro [Ruby® Notes for Professionals book]. Seu download é gratuito mas podes gentilmente pagar um cafézinho para o autor origina. Quanto a mim, me contento com ganhar uma estrelinha neste repositório.
Este livro Ruby® Notes for Professionals foi compilado do Stack Overflow, o conteúdo é escrito pelas pessoas bonitas do Stack Overflow, e eu apenas o trazudo como forma de contribuir a comunidade Ruby e aprender. O conteúdo de texto é lançado sob Creative Commons BY-SA. Os creditos ao final.
